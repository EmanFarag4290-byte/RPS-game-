# Neon Rock Paper Scissors 🪨📄✂️

A sleek, modern Rock Paper Scissors game built with vanilla HTML5, CSS3, and JavaScript. Featuring a dark mode aesthetic, smooth animations, and a custom confetti engine.

## 🚀 Live Trial
https://rps-easy-game.netlify.app/

## ✨ Features
* **Animated Scoreboard:** Real-time tracking of wins, losses, and draws.
* **AI "Thinking" State:** Simulated CPU processing for a more immersive feel.
* **Responsive Design:** Works perfectly on mobile, tablet, and desktop.
* **Custom Visuals:** Pure CSS radial gradients and a Canvas-based confetti system.

## 🛠️ Built With
* HTML5
* CSS3 (Custom Properties & Keyframes)
* JavaScript (ES6+)
* Google Fonts (Boogaloo & Nunito)
